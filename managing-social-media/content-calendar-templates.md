## Content Batch Planning

### Monthly Batch Session Template

**Session 1: Planning (1-2 hours)**
- Review last month's analytics
- Identify top performing content
- Plan themes for upcoming month
- Create content ideas list
- Assign to calendar

**Session 2: Writing (2-3 hours)**
- Write all captions for month
- Create hashtag sets
- Draft email copy if needed
- Write video scripts

**Session 3: Design (2-3 hours)**
- Create templates in Canva
- Design all graphics
- Create carousel slides
- Export all assets

**Session 4: Video (2-3 hours)**
- Film all video content
- Record voiceovers
- Edit in CapCut/Canva
- Export final versions

**Session 5: Scheduling (1 hour)**
- Upload to scheduling tool
- Set post times
- Double-check all content
- Prepare engagement prompts

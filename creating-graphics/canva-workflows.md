## Export Settings

### For Web
- **Format**: PNG or JPG
- **Quality**: High (but reasonable file size)
- **Color Profile**: sRGB
- **Size**: Optimized for intended use

### For Social Media
- **Format**: PNG (graphics) or JPG (photos)
- **Quality**: High
- **Size**: Platform-specific dimensions
- **Animation**: GIF or MP4 if animated

### For Print
- **Format**: PDF Print
- **Color Profile**: CMYK (if available)
- **Include Bleed**: Yes (3mm/0.125in)
- **Flatten PDF**: For compatibility
- **Resolution**: 300 DPI

### For Presentations
- **Format**: PPTX or PDF
- **Quality**: High
- **Include Notes**: If needed
- **Link Slides**: For interactive PDFs

## Keyboard Shortcuts

### General
- `Ctrl/Cmd + S` - Save
- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Shift + Z` - Redo
- `Ctrl/Cmd + C` - Copy
- `Ctrl/Cmd + V` - Paste
- `Ctrl/Cmd + D` - Duplicate
- `Delete/Backspace` - Delete selected

### Elements
- `Ctrl/Cmd + G` - Group
- `Ctrl/Cmd + Shift + G` - Ungroup
- `Ctrl/Cmd + ]` - Bring forward
- `Ctrl/Cmd + [` - Send backward
- `Ctrl/Cmd + Shift + ]` - Bring to front
- `Ctrl/Cmd + Shift + [` - Send to back

### View
- `Ctrl/Cmd + 0` - Fit to screen
- `Ctrl/Cmd + 1` - Actual size
- `Ctrl/Cmd + +` - Zoom in
- `Ctrl/Cmd + -` - Zoom out

### Text
- `Ctrl/Cmd + B` - Bold
- `Ctrl/Cmd + I` - Italic
- `Ctrl/Cmd + U` - Underline
- `Ctrl/Cmd + Shift + L` - Align left
- `Ctrl/Cmd + Shift + C` - Align center
- `Ctrl/Cmd + Shift + R` - Align right

## Pro Tips

### Design Faster
1. Use `/` to quick-search elements
2. Create reusable templates
3. Use Brand Kit for consistency
4. Duplicate pages for variations
5. Use Smart Mockups for presentations

### Better Designs
1. Use alignment guides (they snap automatically)
2. Group related elements
3. Use consistent spacing (hold Shift to measure)
4. Lock background elements
5. Use transparency for layering

### Team Collaboration
1. Share folders for organized assets
2. Use comments for feedback
3. Track version history
4. Create template libraries
5. Set brand kit permissions

## Common Tasks

### Creating a Social Media Set
1. Start with one platform design
2. Use "Resize" to create other sizes
3. Adjust layouts for each format
4. Maintain brand consistency
5. Export all at once

### Designing a Presentation
1. Choose template or start blank
2. Set master slide with brand elements
3. Create consistent slide layouts
4. Add content with hierarchy
5. Include visual breaks
6. End with clear CTA

### Making Print-Ready Files
1. Start with correct dimensions + bleed
2. Keep important content in safe zone
3. Use high-res images (300 DPI)
4. Convert to CMYK if possible
5. Export as PDF Print with bleed marks

### Quick Social Media Content
1. Browse templates for inspiration
2. Swap in brand colors
3. Replace placeholder text
4. Add/swap images
5. Export and post

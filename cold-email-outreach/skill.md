# Cold Email Outreach Specialist

## Golden Rules
1. **Personalization > Volume** - 50 personalized emails beat 500 generic
2. **Value First** - Give before you ask
3. **One CTA** - Make it easy to respond
4. **Follow Up** - 80% of deals happen after 5+ touches
5. **Respect the Inbox** - Quality over quantity

## Email Frameworks

### AIDA Framework
```
Attention: Hook them in line 1
Interest: Connect to their problem
Desire: Show the solution/benefit
Action: Clear, easy CTA
```

### PAS Framework
```
Problem: Identify their pain
Agitate: Make it feel urgent
Solution: Present your offer
```

### BAB Framework
```
Before: Current painful state
After: Desired future state
Bridge: How you get them there
```

### QVC Framework (Quick Value CTA)
```
Quick context: Why you're reaching out
Value prop: What's in it for them
CTA: Simple next step
```

## Email Templates

### Template 1: The Compliment Opener
```
Subject: Loved your [specific content]

Hi [Name],

I just read your [article/post/talk] about [topic] - particularly the part about [specific point]. Really resonated with me because [why].

I work with [similar companies/people] who face [related challenge], and I thought [brief value prop].

Would you be open to a quick chat to see if this might help [their company] too?

[Your name]
```

### Template 2: The Mutual Connection
```
Subject: [Mutual connection] suggested I reach out

Hi [Name],

[Mutual connection] mentioned you might be a great person to talk to about [topic].

I help [target audience] with [specific problem], and given [their company]'s focus on [relevant area], thought there might be some synergy.

Would a 15-minute call this week make sense?

[Your name]

P.S. [Mutual] says hi!
```

### Template 3: The Trigger Event
```
Subject: Congrats on [trigger event]

Hi [Name],

Saw the news about [specific event - funding, hire, launch, etc.] - congrats!

When companies hit this stage, they often start dealing with [related challenge]. We've helped [similar company] solve this by [brief solution].

Worth a quick conversation to see if we could help [their company] too?

[Your name]
```

### Template 4: The Problem-Agitate
```
Subject: [Pain point] at [Company]?

Hi [Name],

Quick question: is [specific problem] something [their company] is dealing with right now?

I ask because most [their role] I talk to mention [problem] as a major time sink, especially when [specific scenario].

We built [solution] specifically to fix this - [specific result] for companies like [similar company].

Mind if I send over a quick 2-min video showing how it works?

[Your name]
```

### Template 5: The Value-First
```
Subject: [Resource] for [their company]

Hi [Name],

I put together [specific resource - guide, analysis, template] that I thought might be useful for [their company].

[One sentence about what it covers and why it's relevant to them]

Here's the link: [URL]

No strings attached - just thought it might help. If you find it useful and want to chat about how we help companies like [theirs] with [problem], I'm happy to jump on a quick call.

[Your name]
```

### Template 6: The Breakup Email
```
Subject: Should I close your file?

Hi [Name],

I've reached out a few times but haven't heard back - totally understand you're busy.

I'll assume the timing isn't right and close your file for now.

If [problem] becomes a priority later, feel free to reach out. I'm always happy to help.

Thanks for your time,
[Your name]

P.S. If I'm way off base and you'd still like to chat, just reply "interested" and I'll send over some times.
```

## Follow-Up Sequences

### Standard 5-Touch Sequence
```
Email 1 (Day 0): Initial outreach - personalized, value-focused
Email 2 (Day 3): Different angle + "bumping this up"
Email 3 (Day 7): Value-add (share relevant content/resource)
Email 4 (Day 14): Social proof (case study, testimonial)
Email 5 (Day 21): Breakup email
```

### Follow-Up Templates

**Follow-Up #2 (Different Angle)**
```
Subject: Re: [Original subject]

Hi [Name],

Wanted to make sure this didn't get buried.

Another thought: [different angle or additional value point].

Would [specific day] work for a quick call?

[Your name]
```

**Follow-Up #3 (Value Add)**
```
Subject: Thought you'd find this useful

Hi [Name],

Came across this [article/resource/data] and thought of [their company]:

[Link or brief summary]

[One sentence connecting it to their situation]

Still happy to chat if [original offer] is of interest.

[Your name]
```

**Follow-Up #4 (Social Proof)**
```
Subject: How [Similar Company] solved [problem]

Hi [Name],

Quick case study I thought might resonate:

[Similar company] was dealing with [same problem]. After implementing [solution], they saw [specific result].

I'd love to show you how we could do something similar for [their company].

Worth 15 minutes?

[Your name]
```

## Reply Handling

### Positive Reply
```
1. Respond within 1 hour (max 4 hours)
2. Thank them briefly
3. Propose 2-3 specific times
4. Include calendar link (Calendly)
5. Confirm timezone
```

**Template:**
```
Great to hear from you, [Name]!

How does [Day] at [Time] or [Day] at [Time] work?

Or feel free to grab a time that works best: [Calendly link]

Looking forward to it.
```

### Objection: Not Interested
```
Thanks for letting me know, [Name].

Mind if I ask - is it the timing, or is [problem] just not a priority right now?

Either way, I appreciate you taking the time to respond.
```

### Objection: Too Busy
```
Totally understand - [time of year/quarter] is crazy.

Would it make sense to reconnect in [timeframe]? Happy to circle back then.

In the meantime, here's [resource] in case it's useful.
```

### Objection: Using Competitor
```
Good to know you're already focused on [problem area].

Out of curiosity, how's that going? Any gaps you've noticed?

We often work alongside [competitor] or help companies who've outgrown them. No pressure either way - just curious.
```

### Objection: Send More Info
```
Absolutely - here's a quick overview: [link to one-pager/video]

The TL;DR: [one sentence value prop]

Happy to answer any questions. Would a quick 10-minute call be easier than email back-and-forth?
```

### Referral Request (If Not Right Person)
```
Thanks for letting me know, [Name].

Who on your team typically handles [problem area]? Would love an intro if you think it makes sense.

Appreciate your help!
```

## Deliverability for Cold Email

### Separate Domain Setup
```
1. Buy similar domain (if main is company.com, use getcompany.com, trycompany.com)
2. Set up Google Workspace or Outlook
3. Configure SPF, DKIM, DMARC
4. Warm up for 2-4 weeks before sending
```

### Warmup Process
```
Week 1: 10-20 emails/day (to real contacts who reply)
Week 2: 30-40 emails/day
Week 3: 50-75 emails/day
Week 4: 75-100 emails/day

Tools:
- Instantly.ai (built-in warmup)
- Warmbox.ai
- Lemwarm
```

### Sending Limits
```
Gmail: 500/day (workspace)
Outlook: 300/day
Cold email tools: Follow their guidelines

Best Practice: Stay under 100 cold emails/day per inbox
```

### Avoid Spam Triggers
```
✗ Link-heavy emails
✗ Image-only emails
✗ Attachments in cold emails
✗ URL shorteners (bit.ly)
✗ Spam words (see main email marketing skill)
✗ Too many follow-ups too fast

✓ Plain text or minimal HTML
✓ 1-2 links max
✓ Personalization
✓ Custom tracking domain
✓ Proper unsubscribe (for scale)
```

## Metrics & Optimization

### Cold Email Benchmarks
| Metric | Target | Red Flag |
|--------|--------|----------|
| Open Rate | 40-60% | <30% |
| Reply Rate | 5-15% | <3% |
| Positive Reply | 2-5% | <1% |
| Meeting Booked | 1-3% | <0.5% |
| Bounce Rate | <3% | >5% |

### A/B Testing Priorities
```
1. Subject lines (biggest impact)
2. Opening line
3. CTA
4. Email length
5. Send time
6. Personalization depth
```

### Weekly Review Questions
```
1. Which subject lines had highest opens?
2. Which templates got most replies?
3. What objections keep coming up?
4. Which segments convert best?
5. What's my reply-to-meeting ratio?
```

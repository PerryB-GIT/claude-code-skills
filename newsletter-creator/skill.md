## Writing Guide

### Headlines/Subject Lines
```
# Formulas
- [Number] + [Adjective] + [Noun] + [Promise]
  "7 Brutal Truths About Startup Fundraising"

- How to [Achieve Goal] Without [Pain Point]
  "How to 10x Your Productivity Without Burning Out"

- The [Adjective] Guide to [Topic]
  "The Lazy Person's Guide to Building Wealth"

- What [Authority] Knows About [Topic] That You Don't
  "What Top VCs Know About Pitching That You Don't"

- Why [Counterintuitive Statement]
  "Why Working Less Makes You More Successful"
```

### Opening Lines
```
# Hook Types
- Contrarian: "Everything you've heard about X is wrong."
- Story: "Last Tuesday, I made a $50K mistake."
- Question: "What would you do with an extra 10 hours/week?"
- Stat: "73% of marketers are getting this wrong."
- Direct: "Today I'm sharing my exact playbook for X."
```

### Transitions
```
- "Here's the thing..."
- "But here's where it gets interesting..."
- "Which brings me to..."
- "Now, you might be thinking..."
- "The bottom line:"
- "Speaking of which..."
```

### CTAs
```
# Engagement CTAs
- "Hit reply and tell me..."
- "What's your take? Reply and let me know."
- "I'd love to hear your thoughts on this."

# Sharing CTAs
- "Know someone who'd find this useful? Forward this along."
- "Share this with a friend who needs to see this."

# Action CTAs
- "Download the template here: [link]"
- "Get the full guide: [link]"
- "Join the waitlist: [link]"
```

## Design & Formatting

### Email Width
```
- Max width: 600px
- Text width: 500-550px
- Mobile-first design
```

### Typography
```
- Font: System fonts for reliability
  Arial, Helvetica, sans-serif
  Georgia, Times, serif

- Size: 16-18px body, 24-32px headers
- Line height: 1.5-1.6
- Paragraph spacing: 16-24px
```

### Visual Elements
```
# Use Sparingly
- Headers with emojis
- Horizontal dividers
- Blockquotes for emphasis
- Bullet points for lists
- Bold for key points

# Avoid
- Too many images (deliverability)
- Background images (email client support)
- Complex layouts
- Animated GIFs (file size)
```

### Plain Text vs HTML
```
# Plain Text Benefits
- Higher deliverability
- Feels personal
- Works everywhere
- Easier to write

# HTML Benefits
- Brand consistency
- Visual hierarchy
- Clickable buttons
- Analytics (tracking pixels)

# Recommendation
- Personal newsletters: Plain text or minimal HTML
- Brand newsletters: Clean, simple HTML
```

### Sponsorship Pricing
```
# CPM Model (Cost Per Thousand)
- General: $20-50 CPM
- Niche B2B: $50-100+ CPM
- Premium audiences: $100-200+ CPM

# Example
10,000 subscribers x $50 CPM = $500/placement

# Sponsorship Types
- Primary sponsor: Full ad placement
- Secondary: Smaller mention
- Classified: Text-only link
- Dedicated: Full email (rare)
```
